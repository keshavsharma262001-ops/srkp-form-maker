A flexible and custom form builder and submission capture system for WordPress.

== Description ==

**SRKP Form Maker** is a lightweight, customizable form builder plugin designed to capture front-end form submissions (Contact, Booking, Banquet) and save them to a custom post type.

Features:
* Admin Forms Builder to customize form fields
* Sortable form fields via drag & drop
* Automatic capture and saving of form submissions as custom posts
* Multi-recipient email notifications
* Shortcodes for easy frontend form rendering
